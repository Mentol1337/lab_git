# Лабораторна робота 3

## Опис
Проєкт на C++, який емулює роботу автосервісу. Програма створює заявку на ремонт автомобіля та обчислює вартість послуг.

## Структура проєкту
- src/main.cpp
- src/Car.h
- src/Car.cpp
- src/Service.h
- src/Service.cpp
- Makefile
- Dockerfile
- docker-compose.yml

## Компіляція
make

## Запуск
./app

## Очищення
make clean
