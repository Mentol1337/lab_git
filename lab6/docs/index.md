# Документація до проєкту

## Загальна інформація
Цей репозиторій створено для виконання лабораторних робіт з дисципліни, пов’язаної з використанням Git та GitHub.

## Структура репозиторію
У репозиторії містяться файли лабораторних робіт та супровідна документація.
Для лабораторної роботи №6 була створена папка `lab6`, у якій знаходиться папка `docs` з файлом `index.md`.

## Мета документації
Документація створена для короткого опису репозиторію, його структури та призначення файлів.

## Використані технології
- Git
- GitHub
- Markdown
- GitHub Pages

## Автор
Студент виконав лабораторну роботу та налаштував публікацію документації через GitHub Pages.
